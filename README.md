# <center>一言服务端</center>
  
一言api服务端

支持nodejs及cloudflare work（由大模型生成，人工微调代码）  
目前仅开启心跳引擎一言

## 预览

[在线api（cloudflare work）](http://heartbeat-engine-api.xt-url.com/)

## 安装与更新

假设你已有`git` `node` `pnpm`环境  
如没有可自行安装相关软件

### 下载代码

``` bash
git clone --depth=1 https://github.com/xiaotian2333/one-word-server
```

### 安装依赖

``` bash
pnpm install
```

### 运行

``` bash
pnpm start
```
